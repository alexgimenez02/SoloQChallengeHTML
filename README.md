# SoloQChallengeHTML

Interactive HTML to add at OBS for Aitoreta's [Solo Q Challenge](https://twitter.com/Aitoreta/status/1625221824970629137?s=20&t=a8EjL-11yCRm3t-3sd-DSA)


# Usage

 1. Open the HTML file into an explorer
 2. Use left or right arrow to change characters
 3. Press 'r' button to go back to previous checkpoint
